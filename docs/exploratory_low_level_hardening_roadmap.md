# Exploratory Low-Level Hardening Roadmap

This roadmap tracks speculative low-level hardening themes that are not practical near-term implementation work for this stack.

## Scope

Exploratory topics include:

- side-channel resistance themes
- glitching and physical fault-injection resilience concepts
- speculative-execution-class mitigation research themes
- hardware-adjacent hardening ideas that exceed current runtime/control-plane scope

## Guardrails

- This page is roadmap-only and non-release.
- Do not convert these topics into release blockers without concrete, testable implementation plans.
- Keep immediate product hardening work in active engineering and security backlogs.

## Review Cadence

- Review during security roadmap and architecture planning cycles.
- Promote only when feasibility, ownership, and verification criteria are clear.
