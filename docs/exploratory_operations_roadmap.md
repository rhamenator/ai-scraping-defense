# Exploratory Operations Roadmap

This roadmap is a non-release bucket for speculative operations ideas that should remain visible without entering the active release-blocker queue.

## Scope

The items in this bucket are explicitly exploratory and not current release commitments:

- quantum-oriented operational scenarios
- space/remote-site operational scenarios
- neuromorphic and non-traditional compute operations
- robotics-assisted operations and response automation
- AR/VR-assisted operator workflows
- voice-driven operations interfaces
- IoT-scale defensive operations
- autonomous-systems operations controls

## Guardrails

- Do not treat this page as a release checklist.
- Do not create release blockers from these topics without a scoped design and implementation plan.
- Keep active release work tracked in `release_blockers.md` and the release checklist.

## Review Cadence

- Revisit this bucket during roadmap planning and architecture review cycles.
- Promote items only when they are codifiable, testable, and aligned with near-term product scope.
