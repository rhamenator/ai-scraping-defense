# Exploratory Advanced Cryptography Roadmap

This roadmap tracks research-heavy cryptography themes that are not current implementation work for this stack.

## Scope

Exploratory topics include:

- quantum and post-quantum cryptography planning themes
- advanced privacy-preserving cryptography concepts
- searchable, functional, identity-based, threshold, and similar research-heavy models
- future cryptographic approaches that are not yet practical near-term engineering tasks here

## Guardrails

- This page is roadmap-only and non-release.
- Do not create release blockers from these items without scoped design and validation.
- Promote only concepts that become practical, testable, and operationally supportable.

## Review Cadence

- Review in periodic security and architecture planning cycles.
- Track migration candidates separately when standards and implementation readiness mature.
